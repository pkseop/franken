    한전KPS 원격근무 보안관리 강화에 따라, 신청자 소속부서의 장(처장·소장·지점장·팀장 등)의 승인 및 회사 정보보안 담당자의 확인 후에 계정이 발급되오니 VPN 사용 희망일 최소 2~3일 전에 신청하여 주시기 바랍니다.
