eVPN 사용 환경

O/S
Windows XP/Vista/7 지원

Web Browser
Internet Explorer 6.0 / 7.0 / 8.0

인터넷환경
전용선, ADSL, Wi-Fi, WiBro, HSDPA, 케이블모뎀 등

eVPN 접속 방법

- 사용자 PC의 웹 브라우저(Internet Explore)를 실행하여 주소창에 “https://해당시도URL”을 입력하여 접속을 시도합니다.

※ 접속주소 예) 서울 - https://evpn.sen.go.kr, 경북 - https://evpn.gbe.kr

- 최초 접속 시 ActiveX컨트롤을 이용하여 자동으로 보안프로그램 다운로드 되고 PC에 설치됩니다.
- 정상적으로 보안프로그램 설치가 완료되면, 로그인 화면이 표시됩니다.

eVPN 로그인/로그아웃 방법

로그인 방법
- 이용자 ID/PW, 인증서 패스워드 입력하여 로그인 성공하면 접근 가능한 업무목록(즐겨찾기) 화면이 표시됩니다.

※ 차세대나이스업무에서 신청한 ID/PW 및 업무포털 접속시의 인증서를 사용해야 합니다.

로그아웃 방법
- eVPN 접속 웹 창을 닫으면 자동으로 로그아웃 됩니다.

※ 즐겨찾기 화면에서 로그아웃 메뉴를 클릭하여 로그아웃 할 수 있으며, 비정상종료 등의 상황에서는 바탕화면 우측하단
    작업표시줄의 SecuwaySSL아이콘(  )에서 Exit을 클릭하여 로그아웃 가능합니다.

※ 로그인시에 윈도우즈 방화벽 혹은 바이러스 백신의 개인방화벽에 의해 접속이 차단될 수 있으므로, 방화벽 기능을
    일시 해지하고 사용해야 될 수도 있습니다.

eVPN 패스워드를 잊어버렸을 경우

eVPN 패스워드를 잊어버렸을 경우 VPN 사용자 승인 프로세스에 따라 신규 신청해야 합니다.
- 개인정보 유출, 불법해킹 등의 우려로 인해 외부에서는 패스워드 재발급/초기화등의 프로세스를 별도 제공하지 않으므로 패스워드 관리에 유의하셔야 됩니다.
